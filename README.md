## Overview

Mobile app for AgentPay - create AI agents, manage spending limits, and approve payments on the go.

## Features

- **Wallet Connection** - Connect Freighter wallet
- **Agent Management** - Create, pause, resume agents
- **Real-time Notifications** - Payment alerts
- **QR Code Scanning** - Quick agent adding
- **Payment Approval** - Approve large transactions

## Screens

| Screen | Description |
|--------|-------------|
| Connect | Wallet connection |
| Dashboard | Agent list and stats |
| Create Agent | New agent form |
| Agent Details | View and manage agent |
